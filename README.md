Connect VL53L0X with Development Boards such as ESP32

# VL53L0X       ESP32

- VCC           VCC / 3.3V
- GND           GND
- SCL           IO16
- SDA           IO17
- GPIO1         IO21
- XSHUT         IO22

- Load vl53l0x_add_i2C_pins.ino.
